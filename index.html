隐私政策

生效日期：2025年12月29日

本隐私政策适用于由陈桥（Chen Qiaojun）开发的自动化工具 “ChenQiao RPA”（以下简称“本应用”）。

1. 我们如何使用您的 eBay 数据？

本应用仅通过 eBay 官方 API 获取授权卖家的订单数据，用于以下目的：

  • 同步订单信息到内部系统
  • 自动化订单处理与物流跟踪
  • 提升订单履约效率

2. 我们是否存储您的个人数据？

我们仅在必要期间临时缓存订单相关数据（如订单号、商品信息、配送地址），用于完成自动化流程。我们不会：

  • 出售或共享您的数据给任何第三方
  • 将数据用于营销或广告目的

3. 数据安全

我们采取合理的技术和组织措施保护您的数据，防止未经授权的访问、泄露或丢失。
